Welcome to the Legume Information System Jekyll Theme

Jekyll is a static site generator that takes text written in markup and uses layouts to create a static website. Layouts and their look and feel are provided by a theme. This project is a Jekyll theme created for the Legume Information System (LIS) and provided as an open-source resource for other biological information systems.

Styling

The LIS Jekyll theme uses the UIkit CSS framework. Thus, any markup from UIkit can be used in user-defined markdown content.
