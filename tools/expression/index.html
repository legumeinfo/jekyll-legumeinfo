Expression profile, phylogenetic tree, and coexpression network for a single gene.

Comparing Expression profiles of multiple genes
From the CoNekT Compare Profiles page, you can compare the expression of up to 50 genes on the same graph in each experimental condition or tissue. To use this tool:
  • Change the species to Glycine max
  • Enter a list of gene identifiers, using the prefixed gene IDs, e.g. glyma.Wm82.gnm2.ann1.15g026400
  • Click "Show profiles"

Create a custom heatmap with multiple genes
From the CoNekT Create Custom Heatmap page, you can generate a detailed heatmap for a single species (Default tab) and a cross-species comparative heatmap (Comparative tab). To use this tool, enter a list of gene identifiers, using the prefixed gene ID, e.g. glyma.Wm82.gnm2.ann1.15g026400
