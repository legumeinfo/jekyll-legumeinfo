Download

LIS Datastore

All of the data housed by the Legume Information System are available as downloadable files at the LIS Datastore.

The Datastore is organized by Genus and species, with data collections under species directories for:

/genomes/
genome assemblies (FASTA files)
/annotations/
genome assembly annotations (GFF and FASTA files)
/expression/
gene expression experiments (tab-separated files)
/genetic/
genetic experiments (GWAS, QTL mapping, etc. in tab-separated files)
/markers/
genetic marker mappings to genome assemblies
/synteny/
syntenic associations between different species and accessions (GFF files)
/diversity/
population diversity data (VCF files)

Each collection contains a README file which describes that collection.
